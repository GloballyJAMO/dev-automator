# dev-automator
![GitHub last commit](https://img.shields.io/github/last-commit/globallyjamo/dev-automator) ![License](https://img.shields.io/badge/license-MIT-blue)

A powerful CLI tool designed to automate GitHub workflows, boost developer profiles, and seamlessly unlock GitHub achievements via automated commits and PRs. Built by [@globallyjamo](https://github.com/globallyjamo).

## Usage
```bash
# 1. Make scripts executable
./scripts/setup.sh

# 2. Run the master unlocker script
./scripts/unlock-all.sh

# 3. Or run the interactive Node.js CLI
npm start
```
